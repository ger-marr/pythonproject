Instrucciones de Git para dummies

  • 1. git clone: clonas el repositorio desde github
  • 2. git add: añades los archivos que quieras ira commit
  • 3. git commit -m: Añades a un estado de commit el archivo. (Nota: puedes agregar le comando git -am y es la suma de "add" màs "commit"
  • 4. git push: lanzas el archivo a Github
  • 5. git status: Checas el status en el que están tus archivos. Por ejemplo, puedes ver si los archivos están listos para ser commit
  • 6. git pull: copias la versión más reciente del archivo que se encuentra en github
  • 7. git checkout master: creas un nuevo branch sin afectar lo que ocurre enel archvio master. (Ej: git checkout feature, crea un nuevo branch llamado feature)
  • 8. git merge feature: En este ejemplo en particular, tendrìas que estar en el branch master, y hacer merge del branch feature.
  • 9. git branch: Puedes ver el branch que esta corriendo
